# CustomBox

Projet réalise à l'issue du crazy charly day 2022. Notre objectif premier était de se focaliser plutôt sur la partie front end que la partie back end.
En terme de back end la connexion et l'inscription ont été réalise. De plus l'accueil à été adapté pour permettre une bonne visualisation sur webetu.
En terme de front end toute la partie inscription, connexion, a propos ont été réalisées.Tous les formulaires ont été réadapté pour permettre une bonne expériene visuelle pour l'utilisateur.
